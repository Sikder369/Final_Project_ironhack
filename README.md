# RetailIQ: Advanced E-Commerce Intelligence Dashboard

## Overview

RetailIQ is an end-to-end e-commerce analytics project built using the Olist Brazilian E-Commerce Dataset. The project combines data cleaning, SQL analysis, exploratory data analysis (EDA), customer segmentation, statistical testing, and interactive dashboards to generate actionable business insights.

## Objectives

* Analyze sales performance and revenue trends
* Evaluate delivery efficiency and customer satisfaction
* Identify top-performing product categories
* Perform customer segmentation using RFM analysis
* Support data-driven business decision-making

## Tools & Technologies

* Python
* Pandas & NumPy
* SQLite3
* Plotly
* Streamlit
* Tableau

## Key Features

* Interactive Streamlit Dashboard
* Executive KPI Monitoring
* Customer Segmentation (RFM)
* Delivery Performance Analysis
* Hypothesis Testing
* Tableau Business Intelligence Dashboard

Olist Brazilian E-Commerce Public Dataset
